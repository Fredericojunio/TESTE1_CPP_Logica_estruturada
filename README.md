# TESTE1_CPP_Logica_estruturada
# Descrição do Exercício

Este projeto consiste na realização de exercícios em C++, com o objetivo de praticar e consolidar conhecimentos fundamentais da linguagem. Os exercícios abordam temas como:

- Estruturas de controlo (if, else, switch)
- Ciclos de repetição (for, while, do-while)
- Funções
- Vetores e matrizes
- Manipulação de entrada e saída
- Conceitos básicos de programação orientada a objetos

O projeto visa desenvolver a lógica de programação, a organização do código e a resolução de problemas utilizando C++.
